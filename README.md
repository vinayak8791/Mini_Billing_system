# Mini_Billing_system
🚀 Excited to share my latest project: **Billing System Application**

I have developed a desktop-based Billing System using **Java Swing and JDBC**, which helps manage products, generate bills, and handle cart operations efficiently.

🔹 **Key Features:**
✔️ Product listing from database
✔️ Add to cart with quantity management
✔️ Auto total calculation
✔️ Remove items from cart
✔️ Save bill with database integration

🔹 **Tech Stack:**
💻 Java (Swing)
🗄️ MySQL
🔗 JDBC

This project helped me strengthen my understanding of:

* GUI development in Java
* Event handling
* Database connectivity
* MVC architecture basics
